# Description
Design files and firmware for the pump module driver PCB

## Arduino Firmware Version
Please ensure that the Arduino firmware itself has been updated to at least 3.0.1 for correct functioning of the BLE connection. See [this](https://blog.arduino.cc/2026/03/02/you-can-now-use-wi-fi-and-bluetooth-le-simultaneously-on-arduino-nina-based-boards-heres-how/) article for reference.
